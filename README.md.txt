# Used Car Price & Market Analysis with Python

This project analyzes used car prices using Python (Pandas, Matplotlib, Seaborn).

Dataset: [Used Car Price Dataset – Kaggle](https://www.kaggle.com/datasets/rishabhkarn/used-car-dataset)

## Folder structure
- `data/`: contains raw `used_cars.csv` and cleaned `used_cars_cleaned.csv`.
- `notebooks/eda_used_cars.py`: main Python script for cleaning and plotting.
- `plots/`: saved PNG plots (histograms, boxplots, etc.).

## Key insights
- Luxury car brands have higher average resale prices.
- Diesel cars tend to be more expensive than petrol cars.
- Price decreases with car age and higher kilometers driven.
